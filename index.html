WGDev - WebGUI comand line utilities

Getting Started with WGDev

Install

  • $ curl -O http://haarg.org/wgd
  • $ chmod +x wgd

Read Intro Docs

  • $ ./wgd intro

Developing for WGDev

Check out code

  • $ git clone git://github.com/haarg/wgdev

Create new command

  • $ cp ex/Command.pm lib/WGDev/Command/MyNewCommand.pm
  • $ vim lib/WGDev/Command/MyNewCommand.pm

Build new executable

  • $ dzil build
